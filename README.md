# Fun With Robots — Course Website

Plain HTML/CSS/JS site for the course 98-012 Fun with Robots.

## Structure
- Home: `index.html`
- About: `about.html`
- Schedule: `schedule.html`
- Assets: `assets/`

## Run Locally
Use any static server. Example (Python):

```bash
python3 -m http.server 8000
# then visit http://localhost:8000/
```

Or use VS Code Live Server / GoLive extensions.
